# Viajes Chile 🏔️

Landing page para la agencia de turismo ficticia **Viajes Chile**, desarrollada como prueba de evaluación del módulo de Fundamentos del Desarrollo Front-end de Desafío Latam.

## Demo en línea

🔗 [https://pjaram82-png.github.io/viajes-chile/](https://pjaram82-png.github.io/viajes-chile/)

## Descripción

La página presenta a la agencia de viajes con un diseño moderno, oscuro y responsivo, invitando a descubrir distintos destinos turísticos de Chile: Torres del Paine, el desierto de Atacama, la Isla de Pascua y la Araucanía.

## Secciones

- **Navbar** fija con smooth scroll a cada sección
- **Carousel** con imágenes destacadas de destinos chilenos
- **¿Quiénes somos?** con favicons y 3 párrafos (oculta en pantallas pequeñas)
- **Destacados** con 4 cards de destinos turísticos
- **Formulario de contacto** con validación y tooltip
- **Footer** con links a redes sociales

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (variables, transiciones)
- [Bootstrap 5.3.3](https://getbootstrap.com/) (CDN)
- [Font Awesome 6.5.1](https://fontawesome.com/) (CDN)
- [Google Fonts – Open Sans](https://fonts.google.com/specimen/Open+Sans)
- JavaScript vanilla (componentes Bootstrap JS: Carousel, Tooltip, Collapse)
